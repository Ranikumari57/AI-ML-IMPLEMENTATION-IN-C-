This is a simple Machine Learning project in C++ where the program predicts house price based on size using a basic linear regression model.

## 🎯  Objective of the Project

The main goal of this project is:

- To understand how Machine Learning works in C++
- To build a prediction model without external ML libraries
- To predict house price based on input size
- To implement Linear Regression manually

## 📊  Concept Used: Linear Regression

Linear Regression is a statistical method used to find the relationship between two variables:

- Input (X): House Size
- Output (Y): House Price

## 📌 Formula:

Y= mx+c

Where:
- m = slope (rate of change)
- c = intercept (starting value)
